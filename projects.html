Southbourne

Prayer Room, Emmanuel Church

Emmanuel Church in Southbourne asked for a new room to be created within the main church room. The minister specifically asked for windows & ventilation.

I started by creating two stud walls in between both pillars, insulating and double boarding, then finally taping and jointing all myself. I designed the windows from scratch and installed toughened glass in each one. I’ve installed a Fire door with lining and all ironmongery, then designed and created one cupboard to house a lot of loose instruments and an open shelf unit to store the church’s items. Finally I have painted and decorated everything you can see in the photos.

Parkstone

Roof Balcony

This was a job based in Parkstone which I saw throughout. Due to the previous tenant using the roof as a balcony and the timbers not being structurally appropriate, all existing timber work had to be stripped back. I installed 9x2 joist to a structural drawing supplied, then built a tapped floor with all the posts you can see coming out of the floor before any sealing could be done. After the roof had been sealed using an epoxy resin I returned to install all the cladding which is external and internal. I then painted it all. After all of that I installed the facia, soffit and guttering.
